# choujiang

三步抽奖页面，带后台密钥管理：

1. 选择道具
2. 从七个图标中随机抽取一个
3. 从三个场景中随机抽取一个
4. 展示最终三个结果

## 功能

- 后台登录生成密钥，每个密钥绑定一个老板
- 用户输入密钥后才能抽奖
- 密钥只能使用一次，最终结果提交后自动销毁
- 后台可查看密钥状态和抽奖结果记录

## Linux 一键启动

需要服务器已安装 Node.js 18+。

```bash
chmod +x start.sh
./start.sh
```

默认端口：`9000`

默认后台：

- 地址：`http://服务器IP:9000/admin.html`
- 账号：`admin`
- 密码：`admin123456`

自定义端口和后台密码：

```bash
PORT=9001 ADMIN_PASSWORD='你的强密码' ./start.sh
```

脚本会先检查端口是否占用，占用则结束占用进程，然后启动服务。
